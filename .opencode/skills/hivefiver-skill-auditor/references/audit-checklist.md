# Skill Audit Checklist

## Structural Checks
- Has clear trigger description.
- Has deterministic workflow.
- Includes references/templates/scripts when needed.
- Maps to at least one HiveFiver phase.

## Quality Checks
- Avoids duplicated scope with another skill.
- Last verified date is acceptable for production use.
- Contains clear failure/fallback behavior.

## Inclusivity Checks
- Dev lane covered.
- Marketing lane covered.
- Finance lane covered.
- Office operations lane covered.
- Web research/browsing lane covered.

## Link Hygiene
- Includes canonical links for external dependencies/catalogs.
- Flags unverified third-party sources for manual review.
