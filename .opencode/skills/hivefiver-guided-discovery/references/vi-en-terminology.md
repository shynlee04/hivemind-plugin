> Technical terms are preserved in English; Vietnamese wraps context around them.

## Beginner-Friendly Explanations (Vietnamese)

| Concept | Explanation for L0 Users |
|---------|------------------------|
| Agent | Một "nhân viên ảo" có nhiệm vụ riêng — ví dụ agent kiểm tra code, agent lập kế hoạch |
| Skill | Kiến thức chuyên môn mà agent có thể sử dụng — giống như kỹ năng của con người |
| Command | Một lệnh mà bạn gõ để yêu cầu agent làm việc gì đó — như /hivefiver build |
| Workflow | Trình tự các bước mà agent phải theo — bước 1, bước 2, bước 3... |
| Pipeline | Chuỗi các giai đoạn liên tiếp: tìm hiểu → thiết kế → xây dựng → kiểm tra |
| Gate | Điểm kiểm tra bắt buộc — agent phải vượt qua trước khi tiếp tục |
| Framework | Bộ công cụ và quy tắc mà tất cả agents sử dụng chung |

## Sentence Patterns for Bilingual Questions

```
Pattern 1 (Vietnamese primary):
  [Vietnamese question] ([English technical term])
  Example: "Agent này sẽ làm gì? (mô tả chức năng chính)"

Pattern 2 (Both languages):
  EN: [English question]
  VI: [Vietnamese question]

Pattern 3 (Vietnamese with English anchors):
  "Bạn muốn tạo [asset type] gì? Ví dụ: agent, skill, hoặc command."
```
