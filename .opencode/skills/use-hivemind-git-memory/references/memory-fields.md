# Memory Fields

Use only stable fields in continuity output:
- `intent`
- `decision`
- `rationale`
- `affected_scope`
- `related_commits`
- `activity_type`
- `phase_type`
- `branch`
- `worktree`
- `commit_anchor`
- `confidence`
- `next_step`

Avoid inventing hidden session state or semantic graphs when git evidence is thin.
