# HM Course Correct

## Objective
Repair direction when a session, workflow, or delegation branch has drifted away from its intended path.

## Context
- Arguments: `$ARGUMENTS`
- Trigger: purpose class `course-correction`
- Output focus: diagnosis, re-anchoring, and resumed command routing

## Process
1. Diagnose why the current path drifted or blocked.
2. Re-anchor the required context and continuity surfaces.
3. Recommend the minimum safe next command instead of branching wider.

## Output Contract
- drift_summary
- repaired_context
- resumed_route
- next_command
