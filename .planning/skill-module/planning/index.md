# Skill Module Planning Packets

**Last updated:** 2026-03-19  
**Status:** Initial packet family established

## Purpose

This directory groups planning packets for the skill-module branch. Each packet family should stay focused on one decision lane so later sessions can resume the right lane without re-reading unrelated material.

## Packet Families

| Family | Scope | Status |
|--------|-------|--------|
| `skill-pack-naming/` | Naming, route framing, milestone order, and branch posture | active |

## Rule

Keep packet families narrow. When a branch becomes its own lane, create a separate family instead of overloading the naming packet.
