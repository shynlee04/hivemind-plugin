    3. **Remove confirmed zero-import deps**: Remove each unused dependency from `dependencies` in package.json. Run `npm install` to update lockfile.

    4. **Consolidate YAML libs**: If both `yaml` and `js-yaml` are present and only `yaml` has importers, remove `js-yaml` and consolidate.

    For each removal:
    - Confirm zero imports in src/ (grep)
    - Confirm zero imports in tests/ (grep)
    - Remove the line from package.json
    - Run `npm install` to update lockfile
    - Run `npm run typecheck` and `npm test` to verify

    Do NOT remove dependencies that are:
    - Used in src/ or tests/
    - Peer dependencies (@opencode-ai/plugin, etc.)
    - Build/dev tooling (typescript, vitest, rimraf, etc.)
