## Research-Driven Observations Captured

- Background delegation must be treated as a **subsystem**, not a single tool or single module.
- Hooks, plugin wiring, status retrieval, notification delivery, and persistence/recovery are material parts of the behavior contract.
- PTY/background command execution and semantic agent delegation have different runtime truths and should not be hidden behind a fake single-tool abstraction.
