# Phase 24.3.1 Plan 00: Session Naming Service Summary

Created the canonical session naming service — a pure-logic, zero-dependency TypeScript module that generates and parses machine-parsable OpenCode session titles.

## Key Deliverables

- **`generateSessionTitle(input)`** — produces titles in format `{framework}/{workflow}/{classification}/{agent}/{purpose}@{depth}`
- **`parseSessionTitle(title)`** — reverses generation, validates format, returns `null` for invalid input
- **`NamingInput` / `ParsedNaming`** — typed interfaces for input/output
- **14 unit tests** — cover format correctness, roundtrips, sanitization, edge cases, null returns
