## Plan 01 — sync-assets.js Fixes (Commit `e901fa91`)

**3 critical features added to `scripts/sync-assets.js`:**

| Feature | Description |
|---------|-------------|
| **Backup** | Before `rmSync` on any target directory, all existing entries are backed up (`targetPath + ".backup"`) |
| **Conflict detection** | For each existing file with a matching source, reads both files and warns if content diverges |
| **Command mirror** | After main sync, copies all non-GSD entries from `.opencode/commands/` to `.opencode/command/` with same backup+conflict logic |

**Additional safeguards:**
- `.backup` files are skipped in both main sync and mirror loops to prevent cascading backups
- `gsd-*` files in `.opencode/command/` are never touched
- Binary file reads wrapped in try/catch for safe handling
