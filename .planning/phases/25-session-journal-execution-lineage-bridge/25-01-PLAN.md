| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-25-01 | Tampering | `appendJournalEntry` | mitigate | Require idempotency key matching and append one JSONL record per unique key; tests assert duplicate append suppression. |
| T-25-02 | Information disclosure | `renderJournalEntryMarkdown` | mitigate | Render only explicit `summary`/metadata fields; do not read raw transcripts or continuity records inside renderer. |
| T-25-03 | Elevation of privilege | `.hivemind/` taxonomy | accept | Category docs are not executable and do not grant runtime authority; marker text explicitly says audit/projection only. |
