## Design Decision for DAG Hierarchies

The continuity tree allows children to appear under multiple parents (DAG structure). `rebuildChildToRootMain()` uses a **first-found wins** strategy: it walks the tree breadth-first and assigns the first root encountered for each child. This matches the existing runtime behavior where the first-registered parent is treated as canonical.
