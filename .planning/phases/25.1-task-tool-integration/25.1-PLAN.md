### Wave 3: Trajectory Integration (Depends on Wave 2)

#### Task 3.1: Add Trajectory Creation in `recordChildTaskDelegation()`
**File:** `src/features/session-tracker/tool-delegation.ts`
**Lines:** After 331 (after `projectIndexWriter.addSession()`), before 336 (taskResult block)
**Decisions:** D-01, D-03, D-06, D-07

**What to do:**
1. Import `attachTrajectoryEvidence` and `eventTrajectory` from `../../task-management/trajectory/index.js`
2. After line 331, add trajectory creation logic:
   ```typescript
   // REQ-25.1-01: Trajectory auto-creation on dispatch
   const trajectoryId = `traj-${childSessionID}`
   try {
     attachTrajectoryEvidence({
       projectRoot: this.projectRoot,
       trajectoryId,
       rootSessionId: rootMain,
       sessionId: childSessionID,
       parentTrajectoryId: `traj-${input.sessionID}`,
       evidenceRef: `session-tracker:delegation:${input.tool}:${childSessionID}`,
     })
     eventTrajectory({
       projectRoot: this.projectRoot,
       trajectoryId,
       eventType: "delegation_dispatch",
       summary: `${input.tool} delegation to ${subagentType}: ${description.slice(0, 200)}`,
       evidenceRef: `session-tracker:child-json:${childSessionID}`,
     })
   } catch (err) {
     // D-07: Best-effort — don't fail delegation
     void this.client.app?.log?.({
       body: {
         service: "session-tracker",
         level: "warn",
         message: `[Harness] Trajectory creation failed for delegation ${childSessionID}`,
         extra: { error: err instanceof Error ? err.message : String(err) },
       },
     })
   }
   ```
