<domain>
## Phase Boundary

Wire trajectory event recording and agent-work-contract creation into OpenCode's native task tool lifecycle. Both `task` (native) and `delegate-task` (custom) must automatically create trajectory records and contracts when delegation occurs. The hook point is `ToolDelegation.recordChildTaskDelegation()` in session-tracker, which handles BOTH tools.
