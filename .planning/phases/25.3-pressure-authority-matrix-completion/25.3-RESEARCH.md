## Adding New Entries

Each entry requires:
- `name` — exact match to plugin.ts tool name
- `authority` — read/write/execute/state
- `mutatesState` — true for writers
- `canExecute` — true for command runners
- `stateSurface` — one of 4 taxonomy values
- `pressureBehavior` — preset or custom ToolPressureBehavior
- `evidenceAttachment` — trajectory-ledger/session-journal/execution-lineage/none
- `reason` — short explanation
