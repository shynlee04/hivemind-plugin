## Format Reference

Per D-24-02 and opencode.ai/docs/agents validation, the correct agent frontmatter is:

```yaml
---
description: >
  [One sentence: what this agent does. Second sentence: what artifact it produces.
  Third sentence: when/why the orchestrator calls it.]
mode: all
hidden: true
---
```

**DEPRECATED fields (MUST NOT appear):** `name`, `temperature`, `steps`, `color`, `permission`, `tools`, `depth`, `lineage`, `domain`, `skills`, `instruction`, `reasoningEffort`, `delegation_routing`, `intent_classification`

**Existing format example (gsd-executor.md) for reference:**
```yaml
---
name: gsd-executor
description: Executes GSD plans...
mode: subagent
---
```
