### Non-Destructive Merge Behavior

When `--mode=install` is active, the destructive `rmSync`+`mkdirSync`+`cpSync` pattern is replaced with per-file content comparison:
- Target missing ‚Üí copy from assets
- Target exists, content same ‚Üí skip (no-op)
- Target exists, content differs ‚Üí backup to `.backup`, skip
- Binary/unreadable ‚Üí skip

### Version Drift Optimization

Reads `package.json` version, checks `.hivemind/state/version.json`. If versions match, logs and exits early. Wrapped in try/catch ‚Äî falls through on error. Version stamp written after successful install-mode sync.
