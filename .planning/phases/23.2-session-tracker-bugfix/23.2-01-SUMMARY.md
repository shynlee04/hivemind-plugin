## Changes Made

### Task 1: manifestWriter.addChild() wiring (REQ-23.2-04, Bug #4 HIGH)

Added `this.manifestWriter.addChild(...)` call in `recordChildTaskDelegation()` immediately after the existing `sessionIndexWriter.addChild()` call. The call passes all required delegation context: rootMainSessionID, childSessionID, parentSessionID, delegationDepth, delegatedBy (input.tool), subagentType, and childFile.

**Why unguarded:** The `manifestWriter` property is a required (non-optional) `HierarchyManifestWriter` on the class, and the same file already calls `this.manifestWriter.updateChildStatus()` without a guard at line 343.

### Task 2: Dual-key subagentType extraction (REQ-23.2-05, Bug #5 LOW)

Changed the single-key extraction `args.subagent_type` to a ternary chain that checks `args.subagent_type` first (task tool), then `args.agent` (delegate-task tool), then falls back to `"unknown"`. This fixes the actor field showing "unknown" for delegate-task dispatches.

**Implementation:** Inline ternary using existing `args` Record (via `asRecord()`) — no new type imports needed.
