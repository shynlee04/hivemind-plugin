# CP-DT-01 GAPS — 10 Gaps Analysis & Remediation Plan
> Ngày: 2026-05-18 | Phase: CP-DT-01 Delegate-Task Ecosystem Revamp

## Executive Summary

Sau khi audit toàn bộ codebase delegation và so sánh với spec yêu cầu, xác định **10 gaps** cần khắc phục. Tất cả đều KHẢ THI với SDK @opencode-ai/plugin v1.15.4.

## Gap Inventory

### G1: TUI Thin-Line Injection không hoạt động [CRITICAL]
- **Hiện tại:** `monitor.inject()` chỉ là callback — không append được vào TUI
- **Cần:** Sử dụng `tui.prompt.append` event để inject text vào prompt area của main session
- **SDK:** `EventTuiPromptAppend` type `tui.prompt.append` properties `{ text: string }`
- **Format:** `[DT:{id}] {status} | {elapsed}s | tools={t} bash={b} skills={s} actions={a}`
- **Files:** `monitor.ts`, `plugin.ts` (event hook registration)

### G2: Completion Detection (3 điều kiện) không tồn tại [CRITICAL]
- **Hiện tại:** Không có logic kiểm tra completion
- **Cần:** 3 điều kiện: (1) Tools >1 phút, (2) Assistant last message, (3) File changes
- **SDK:** `session.messages()` parse `Message[]` và `Part[]`
- **Files:** Tạo mới `completion-detector.ts`

### G3: Failure Thresholds không ghi nhận [HIGH]
- **Hiện tại:** Escalation timer fire callbacks nhưng không mark failure/ngừng injection
- **Cần:** 60s(WARN)→120s(NUDGE)→180s(ALERT)→300s(TERMINATE)→600s(final fail)
- **Files:** `escalation-timer.ts`, `monitor.ts`

### G4: Không có Resume/Chain Tasks [HIGH]
- **Cần:** `resume` action → `session.promptAsync()` với cùng session ID; `chain` action → prompt mới
- **Files:** `delegation-status.ts`, `coordinator.ts`

### G5: Không có TUI Completion Message [HIGH]
- **Cần:** Formatted system notification block khi delegate hoàn thành
- **Cơ chế:** Live → `tui.prompt.append`; Ended → `session.prompt({ noReply })`
- **Files:** Tạo mới `notification-formatter.ts`

### G6: Polling không đếm Tool Actions [MEDIUM]
- **Cần:** Đếm tools, bash, skills, actions từ child session messages
- **Files:** `monitor.ts`

### G7: Thiếu Control Tools [HIGH]
- **Cần:** abort, cancel, restart, redirect, adjust-prompt, change-agent
- **Files:** `delegation-status.ts`

### G8: Không tích hợp Session-Tracker [MEDIUM]
- **Cần:** Ghi delegation records vào `.hivemind/session-tracker/`
- **Files:** `delegation-persistence.ts`

### G9: Không có Compact Survival [MEDIUM]
- **Cần:** Đăng ký `experimental.session.compacting` hook
- **Files:** `plugin.ts`, module compact survival mới

### G10: Slot Limit perKeyLimit:2 quá thấp [LOW]
- **Cần:** Tăng perKeyLimit lên 10
- **Files:** `slot-manager.ts`
