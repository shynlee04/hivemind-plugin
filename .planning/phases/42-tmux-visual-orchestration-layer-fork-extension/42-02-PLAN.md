  <action>
    Create `src/features/tmux/integration.ts` with the following exports:

    **`resolveBinary(name: string): Promise<string | null>`**
    - Platform-aware: `which` on POSIX, `where` on win32
    - Uses `execFile` from `node:child_process` (promisified)
    - Returns first line of stdout, trimmed, or null on failure
    - Implementation exactly mirrors the pattern from RESEARCH.md:430-440

    **`getTmuxVersion(tmuxPath: string): Promise<string | null>`**
    - Runs `{tmuxPath} --version`, returns trimmed stdout or null

    **`readPersistedPort(projectDir: string): number | null`**
    - Reads `.hivemind/state/tmux-port.json` from projectDir
    - If file exists: parse JSON, return `port` field if number
    - If file absent: compute deterministic fallback port from sha256 hash of projectDir → 10000..65535 range
    - Import `createHash` from `node:crypto` for deterministic hashing
    - Use `existsSync`/`readFileSync` from `node:fs`, `join` from `node:path`
    - Implementation mirrors RESEARCH.md:451-464

    **`persistPort(projectDir: string, port: number): void`**
    - Ensures `.hivemind/state/` directory exists (mkdirSync recursive)
    - Writes `{ port, updatedAt: Date.now() }` to `.hivemind/state/tmux-port.json`
    - Implementation mirrors RESEARCH.md:466-474

    **`detectServerUrl(projectDir: string): Promise<string | null>`**
    - Try `readPersistedPort(projectDir)` first
    - If port found: return `http://localhost:{port}`
    - If not: return null (caller will detect URL from PluginInput at plugin time)

    **`TmuxIntegration` interface:**
    ```
    export interface TmuxIntegration {
      readonly isAvailable: () => boolean
      readonly version: string | null
      readonly binaryPath: string | null
      readonly opencodeBinaryPath: string | null
      readonly serverUrl: string | null
      readonly projectDirectory: string
    }
    ```

    **`createTmuxIntegrationIfSupported(projectDirectory: string): Promise<TmuxIntegration | null>`**
    - Mirror `createPtyManagerIfSupported()` pattern exactly (RESEARCH.md:226-258):
      1. Resolve tmux binary via `resolveBinary("tmux")` — return null if not found (per D-05 silent fallback)
      2. Check `process.env.TMUX` — return null if not set (we're inside a tmux session)
      3. Resolve opencode binary via `resolveBinary("opencode")` — return null if not found
      4. Detect server URL via `detectServerUrl(projectDirectory)` — null is OK (will be detected from PluginInput later)
      5. Get tmux version via `getTmuxVersion(tmuxPath)`
      6. Return `{ isAvailable: () => true, version, binaryPath, opencodeBinaryPath, serverUrl, projectDirectory }`
    - Wrap in try/catch — any error returns null (silent fallback per D-05)

    All imports from `node:*` modules only — no external dependencies. No Hivemind-specific imports beyond standard Node.
