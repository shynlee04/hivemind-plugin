## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-42-01 | Tampering | config.ts parsing | mitigate | Type-safe validation — copilot boolean strict check, agentLabelFormat string check; no eval/exec from config |
| T-42-02 | Tampering | tmux.ts spawn command | mitigate | Fork's `quoteShellArg()` wraps all arguments in single quotes with `'\''` escape for single-quote injection; config values are shell-quoted |
| T-42-03 | Spoofing | binaryPath config | mitigate | binaryPath is resolved by Hivemind's `which opencode` at bootstrap (uses system PATH), not from user-supplied config — `opencodeBinaryPath` in fork config is OPTIONAL and override-only |
| T-42-SC | Tampering | npm/pip/cargo installs | mitigate | No new packages installed — fork's existing deps unchanged; peer dep pinned to ^1.15.13 |
