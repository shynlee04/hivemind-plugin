<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Continuity → Delegation packet | Malformed continuity record could produce invalid delegation packet |
| Delegation directory → Disk | Concurrent writes could corrupt manifest.json |
| Child session → Delegation packet | Child could modify parent's delegation packet |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-11 | Tampering | writeDelegationPacket | mitigate | Validate all required fields before write; reject packets with missing fields using Zod-like validation |
| T-02-12 | Race Condition | manifest.json updates | mitigate | Read-modify-write with file lock (use `writeFileSync` + `readFileSync` in same function); consider atomic rename pattern (write to .tmp then rename) |
| T-02-13 | Elevation of Privilege | Child session modifying packet | mitigate | Child session can only write to its own packet file (`${childSessionID}.json`); cannot modify parent's packet |
| T-02-14 | Information Disclosure | Delegation packet contents | accept | Packets contain task descriptions and agent info — no PII or secrets; low-value target |
</threat_model>
