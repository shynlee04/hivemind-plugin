## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| runtime output → `.hivemind/state` | Command/delegation/session data may contain secret-like strings before persistence. |
| tool output → event tracker/journal | Public summaries may capture tool output with credentials or tokens. |
| persisted runtime records → status tool | Durable records can be surfaced after in-memory cleanup. |
