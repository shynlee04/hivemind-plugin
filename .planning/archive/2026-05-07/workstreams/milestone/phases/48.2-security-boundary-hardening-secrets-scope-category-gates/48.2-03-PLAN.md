## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| caller session → shared PTY manager | Caller-controlled PTY session IDs cross into shared process-control operations. |
| PTY session inventory → tool response | Global PTY list may reveal sessions from unrelated roots/sessions. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-48.2-03-01 | Information Disclosure | `run-background-command output/list` | mitigate | Require caller-visible delegation ownership before output/list exposure. |
| T-48.2-03-02 | Tampering | `run-background-command input` | mitigate | ask writes to PTY sessions not owned by caller lineage. |
| T-48.2-03-03 | Denial of Service | `run-background-command terminate` | mitigate | ask termination before cancellation marking when caller does not own the PTY session. |
</threat_model>
