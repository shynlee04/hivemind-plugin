<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| OpenCode user/tool call → command tool schema | Untrusted action names, command strings, args, and session IDs enter the harness. |
| Command tool → DelegationManager/PTY runtime | Validated command requests can spawn or control local processes. |
| PTY output → tool response | Process output crosses back to user-visible evidence and may include sensitive content. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-49-01 | Tampering/Elevation | `run-background-command` command parsing | mitigate | Reject implicit shell strings by default or require explicit shell opt-in; test metacharacter command strings before dispatch. |
| T-49-02 | Information Disclosure | PTY output/input/terminate access | mitigate | Preserve caller-visible delegation ownership checks before PTY read/write/terminate; keep foreign-session denial tests. |
| T-49-03 | Repudiation | Successful-looking command start for invalid executable | mitigate | Validate command shape before dispatch and return actionable error envelopes for malformed executable strings. |
| T-49-04 | Denial of Service | Long-running PTY buffers | accept | Existing bounded PTY buffer remains in force; Phase 49 only verifies first-output and offset reliability. |
</threat_model>
