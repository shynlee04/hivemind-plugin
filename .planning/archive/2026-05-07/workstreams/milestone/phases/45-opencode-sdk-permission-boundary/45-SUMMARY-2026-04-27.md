# Phase 45 Summary — OpenCode SDK Permission Boundary

Child session creation no longer sends unsupported `permission` fields. Delegation now builds an explicit prompt-time tool map that allows the resolved agent tools while asking recursive `delegate-task` and `task` usage. Residual remediation replaced the previous broad write-capable fallback with selected-agent policy resolution from live/local agent primitive metadata and a conservative read-only fallback for unknown policy shapes.

## Deviations
