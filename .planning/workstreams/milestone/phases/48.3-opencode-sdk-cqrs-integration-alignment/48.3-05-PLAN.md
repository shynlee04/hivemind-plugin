| Agent tool call → command process | Tool inputs start local processes through manager-owned command dispatch. |
| Caller session → PTY session access | Caller requests output/input/list/terminate for shared PTY sessions. |
| Runtime environment → PTY support | Bun/PTY support can be missing or fail during spawn. |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-48.3-05-02 | Denial of service | PTY unavailable runtime | mitigate | Register explicit no-PTY behavior and route `run` through manager headless fallback. |
| T-48.3-05-03 | Repudiation | Execution mode reporting | mitigate | Tests assert `executionMode` and fallback reason for unavailable/spawn-failure paths. |
</threat_model>
