## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 1 - Bug] False dependency cycles from prose mentions**
- **Found during:** Restart validation.
- **Issue:** The validator treated ordinary prose containing command or agent names as load-order dependencies.
- **Fix:** Dependency edges now require explicit `/command`, `command("name")`, or `@agent` references.
- **Commit:** `7841f870`

**2. [Rule 1 - Bug] Permission narrowing reported as inheritance breakage**
- **Found during:** Restart validation.
- **Issue:** Agent-level denials that safely narrow global allows were reported as contradictions.
- **Fix:** Only attempts to widen a global deny are treated as permission inheritance failures.
- **Commit:** `7841f870`

**3. [Rule 3 - Blocking] Active primitive YAML frontmatter parse failures**
- **Found during:** Restart validation.
- **Issue:** Duplicate YAML keys and unquoted colon-containing descriptions prevented active primitives from loading.
- **Fix:** Removed duplicate keys and converted affected descriptions to valid YAML block scalars.
- **Commit:** `7841f870`
