Make the project-local OpenCode primitive surface restart-valid after Phase 49 stabilized tool contracts. - Runtime validator semantics for dependency edges, permission narrowing, and advisory pipeline overlap.
- Primitive loader schema/discovery behavior for project-local agents, commands, and skills.
- Malformed frontmatter in active symlink-target primitive definitions under `.hivefiver-meta-builder/*/active/refactoring/`.

## Out of Scope

- Live external provider delegation proof; this remains Phase 52 acceptance work.
- Rewriting agent/skill responsibilities beyond syntax/readiness fixes.
- Moving internal runtime state into `.opencode/`; `.hivemind/` remains canonical state root.
