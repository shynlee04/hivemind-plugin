Close three gaps identified in 16.5-GAP-ANALYSIS.md:

1. **CRITICAL-6: Single-Primitive-Per-Call Batch Limitation** — `batchCompile()` accepts an array but each element is a single primitive type. Purpose: Enable atomic mixed-primitive batch operations and add comprehensive eval harness.

Output: Enhanced configure-primitive with mixed-primitive batch, eval harness with metrics, build-time validation scripts.

   interface MixedPrimitiveSpec {
     type: 'agent' | 'command' | 'skill'
     name: string
     spec: AgentSpec | CommandSpec | SkillSpec
   }

   interface MixedBatchResult {
     success: boolean
     results: { type: string; name: string; result: CompileResult }[]
     errors: string[]

     specs: MixedPrimitiveSpec[],
