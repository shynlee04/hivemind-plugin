| Planning docs → future agent context | Untrusted or stale prose may be treated as runtime truth. |
| Guard script → repository files | Script reads repository artifacts and must not mutate runtime state. |
| OpenCode platform evidence → architecture claims | Platform-sensitive claims must cite current evidence before becoming rules. |

</threat_model>
