<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| delegation request → PTY spawn | Untrusted request data is translated into a live child process |
| PTY process → output buffer | Process output is stored and later surfaced to status/reporting APIs |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-16-03 | Tampering | `src/lib/pty/pty-manager.ts` | mitigate | Keep PTY creation behind typed `PtySpawnRequest`, store explicit `cwd` and `env`, and reject unknown session IDs with `[Harness]` errors rather than silently accepting writes |
| T-16-04 | Denial of Service | `src/lib/pty/pty-buffer.ts` | mitigate | Enforce bounded `maxChars` truncation in the PTY buffer so long-running child sessions cannot grow unbounded in memory |
</threat_model>
