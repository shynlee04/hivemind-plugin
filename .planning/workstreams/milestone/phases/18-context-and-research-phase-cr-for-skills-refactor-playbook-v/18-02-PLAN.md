```
If 0 results → EXPOSED: no context mapping. If results → DEFENDED: cite file:line.

### NON-3 (Non-cycles) check:
```
grep -n "entry\|exit\|loop.back\|DONE_WITH_CONCERNS\|BLOCKED\|handoff" .opencode/skills/<name>/SKILL.md
```
If 0 results → EXPOSED: no pipeline stages. If results → DEFENDED or PARTIAL: cite file:line.

### NON-4 (Non-hierarchy) check:
```
grep -n "layer:\|metadata.layer\|NOT for\|exclusion" .opencode/skills/<name>/SKILL.md
```
If no layer in frontmatter AND no exclusion in description → EXPOSED. Otherwise → DEFENDED.

### NON-5 (Non-ecosystem-eval) check:
```
ls .opencode/skills/<name>/evals/ 2>/dev/null
grep -l "stack\|coordinating\|planning" .opencode/skills/<name>/evals/*.md 2>/dev/null
```
If no evals or no stacked scenario → EXPOSED. If stacked evals exist → DEFENDED.

### NON-6 (Non-systematic-pattern) check:
```
wc -l .opencode/skills/<name>/SKILL.md
ls .opencode/skills/<name>/references/ 2>/dev/null | wc -l
ls .opencode/skills/<name>/scripts/ 2>/dev/null | wc -l
```
If body > 500 LOC or body < 30 LOC or unused refs/scripts → EXPOSED. Otherwise → DEFENDED.
```
