# HiveFiver Cross-Domain Skill Discovery

## Purpose
Expand HiveFiver usability beyond software-only workflows while keeping governance quality.

## Domain Packs
- Dev: architecture, implementation, testing, debugging
- Marketing: campaign planning, messaging, content workflows
- Finance: analysis, forecasting, reporting, audit support
- Office Ops: SOP, documentation, process automation
- Web Research: retrieval, synthesis, citation discipline

## Discovery Path
1. Scan local skills with `find-skills` + skill auditor.
2. Fill missing lanes with curated external catalogs.
3. Verify trust, maintenance, and version pinning.
4. Record adoption in skill audit report.
