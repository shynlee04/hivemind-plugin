# Brainstorm Framing

Use this prompt to run a structured ideation session that balances creativity with decision quality.

## Session Flow

1. Frame the challenge as a single problem statement with constraints and success metrics.
2. Generate 8-15 ideas in divergent mode. Do not filter ideas during this phase.
3. Ensure idea diversity across at least three categories (technical, process, product, or business).
4. Group ideas into themes and remove exact duplicates while preserving distinct variants.
5. Define evaluation criteria before scoring (for example: impact, feasibility, risk, time-to-value).
6. Score each idea consistently on a 1-5 scale with short rationale.
7. Select top candidates based on weighted score and strategic fit.
8. Recommend one direction and explain why alternatives were not selected.

## Output Requirements

- Include the problem frame, constraints, and evaluation criteria.
- Provide full idea inventory before final filtering.
- Present top 3 options with tradeoff notes.
- Close with a clear recommendation and first implementation step.
