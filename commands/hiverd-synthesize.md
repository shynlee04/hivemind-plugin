# HiveRD Synthesize

## Objective

Combine multiple research reports, analysis outputs, or evidence sources into a single unified document. Resolve contradictions, identify cross-cutting patterns, and fill coverage gaps.

## Process

1. **Index inputs** — Catalog all input documents with their scope, confidence levels, and key claims.
2. **Extract patterns** — Identify recurring themes, agreements, and contradictions across inputs.
3. **Resolve contradictions** — When sources disagree, present both positions with evidence and recommend a resolution.
4. **Fill gaps** — Identify topics covered by some but not all inputs. Flag missing coverage.
5. **Produce unified output** — Write a coherent synthesis that integrates all inputs.

## Arguments

- `$ARGUMENTS` — Paths to documents to synthesize, or a description of the synthesis goal.

## Output

A unified synthesis report with:
- Input document inventory
- Cross-cutting patterns
- Contradiction resolution log
- Coverage gap analysis
- Unified findings and recommendations
