# HiveMind Dashboard

Launch the interactive Terminal User Interface (TUI) dashboard.

## Usage

```bash
hivemind dashboard [options]
```

## Options

*   --lang <en|vietnamese>: Set initial language (default: en).
*   --refresh <seconds>: Set refresh interval (default: 2).

## Features

*   **Trajectory Pane**: View active session details and hierarchy structure.
*   **Telemetry Header**: Monitor key metrics.
*   **Autonomic Log**: Real-time alerts.
*   **Interactive Controls**: Toggle language, refresh manually, or quit.
