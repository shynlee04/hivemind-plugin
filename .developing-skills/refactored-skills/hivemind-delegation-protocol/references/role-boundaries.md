# Role Boundaries

## Parent Responsibilities
- choose the slice
- set the constraints
- decide whether the child may mutate files
- define the success test and return contract

## Child Responsibilities
- stay inside scope
- report blocked routes instead of improvising new authority
- return evidence before conclusions
- stop when the packet says stop

## Invalid Delegation
- vague "go fix everything"
- recursive handoff with no new boundary
- mixed research, execution, and verification in one child when they need different success tests
- delegation that depends on deleted or phantom skills
