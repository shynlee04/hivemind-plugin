# Code Review Request

## When Mandatory
- After each task in subagent-driven development
- After completing major feature
- Before merge to main

## How to Request
1. Get git SHAs (base + head)
2. Dispatch code-reviewer with: what implemented, requirements, SHAs, description
3. Act on feedback: fix Critical immediately, fix Important before proceeding

## Review Output
### Strengths
### Issues (Critical / Important / Minor)
### Assessment: Ready to merge? [Yes/No/With fixes]
