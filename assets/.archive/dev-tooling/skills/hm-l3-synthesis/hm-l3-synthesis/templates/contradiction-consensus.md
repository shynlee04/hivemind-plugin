# Contradiction + Consensus Template

Use this before final synthesis when sources disagree or evidence quality varies.

| Theme / claim | Evidence sources | Consensus level | Contradictions | Decision | Limitation |
|---|---|---|---|---|---|
| [claim] | [citations] | strong / moderate / weak / none | [summary] | accept / qualify / reject / block | [methodology caveat] |

## Consensus Levels

- **strong:** multiple primary or authoritative sources agree.
- **moderate:** sources agree but are indirect, older, or incomplete.
- **weak:** one primary source or several low-authority sources support the claim.
- **none:** sources conflict or evidence is insufficient.

## Required Final Sections

1. Executive summary.
2. Evidence-backed themes.
3. Contradictions and unresolved questions.
4. Recommendations with rationale.
5. Methodology and limitations.
