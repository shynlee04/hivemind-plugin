# Loading Decision Tree

When you need a platform reference, walk this tree.

## Question: What do you need?

### I need to RUN something
- A command → check `hm-coord-router` first
- A research investigation → `hm-research-deep`
- An audit of the project → `hm-project-audit`
- A code investigation (SCAN/READ/DEEP) → `hm-detective`

### I need to UNDERSTAND something
- How the engine works → `hm-engine-contracts`
- Where state lives → `hm-state-reference`
- What tools are available → `hm-tooling-capability`
- How OpenCode SDK is structured → `hm-platform-opencode`
- How oh-my-openagent was designed → `hm-omo-reference`

### I need to INTEGRATE something
- A third-party SDK → `hm-tech-ingest` (download) + `hm-research-deep` (understand)
- Tech-stack compatibility check → `hm-tech-compliance`
- Subagent delegation patterns → `hm-subagent-patterns`

### I need to COMPRESS something
- Research output → `hm-synthesis`
- Multi-source findings → `hm-synthesis` + `hm-research-chain`

## Anti-pattern
Loading this skill PLUS all 15 references. Each reference loads
~5KB. Total = 75KB wasted context. Load ONE at a time.
