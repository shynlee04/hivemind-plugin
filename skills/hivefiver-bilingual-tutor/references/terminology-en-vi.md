# Terminology EN/VI

- governance -> quản trị
- hierarchy -> phân cấp
- lineage -> chuỗi liên kết
- evidence gate -> cổng bằng chứng
- fallback -> phương án dự phòng
- onboarding -> khởi tạo ban đầu
- requirement distillation -> chưng cất yêu cầu
- contradiction register -> sổ mâu thuẫn
- confidence score -> điểm tin cậy
- task graph -> đồ thị công việc
- user story -> câu chuyện người dùng
