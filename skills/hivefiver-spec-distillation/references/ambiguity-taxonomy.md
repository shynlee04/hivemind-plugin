# Ambiguity Taxonomy

## Scope Ambiguity
- Missing actor
- Missing scope boundary
- Missing success criteria

## Delivery Ambiguity
- Unknown timeline or budget constraints
- Undefined acceptance threshold
- Missing rollout strategy

## Technical/Integration Ambiguity
- Missing data contracts
- Unspecified external dependencies
- Conflicting architecture constraints

## Governance Ambiguity
- Unclear compliance/security requirements
- Missing auditability expectations
- Undefined failure and incident behavior
